## 🎯 核心概念速查

### 认证方式
- JWT Token
- Header: `Authorization: Bearer {token}`
- 有效期: 24小时

### 角色类型
- `student`: 学生
- `admin`: 管理员

### 成果状态
- `pending`: 待审核
- `approved`: 已通过
- `rejected`: 已拒绝

### 成果类型
- `competition`: 学科竞赛
- `paper`: 论文发表
- `patent`: 专利
- `project`: 项目
- `certificate`: 职业证书

### 重要流程

**成果提交**:
1. 上传证书 (`POST /api/v1/student/ocr/recognize`)
2. 确认信息
3. 提交成果 (`POST /api/v1/student/achievements`)

**成果审核**:
1. 获取列表 (`GET /api/v1/admin/achievements`)
2. 审核操作 (`PATCH /api/v1/admin/achievements/{id}/audit`)
